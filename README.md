# 🍰 Engenharia Reversa Gastronômica (E.R.G.) - Skill C.O.D.E.

Este repositório contém a Skill **C.O.D.E. (Confeitaria Ostentação & Design Estrutural) v3.0**. 
Trata-se de um "Super Prompt" (um código de instruções em texto) criado para transformar qualquer Inteligência Artificial em um Cientista de Alimentos de Elite, focado em alta confeitaria com restrições severas (Zero Açúcar, Zero Óleo, Zero Glúten, Low Carb).

---

## 🚀 Como usar isso se eu não sou programador?

Você não precisa instalar nenhum software complexo ou saber programar para usar esta Skill. A IA lê textos, então você só precisa entregar as regras para ela. Escolha um dos métodos abaixo:

### Método 1: O Jeito Mais Fácil (Copiar e Colar no Chat)
Ideal para usar no ChatGPT gratuito, Gemini ou Claude comum.
1. Abra o arquivo [CODE_Skill_v3.md](./CODE_Skill_v3.md) aqui no GitHub.
2. Copie **TODO** o texto que está lá dentro.
3. Abra o seu chat de Inteligência Artificial favorito.
4. Cole o texto inteiro na caixa de mensagem e aperte Enviar.
5. Pronto! A IA vai "vestir" a persona do Cientista de Alimentos e vai começar a te fazer perguntas sobre o que você quer cozinhar.

### Método 2: Instruções Personalizadas (System Prompt)
Se você usa versões pagas ou avançadas (como ChatGPT Plus) que possuem "Custom Instructions" (Instruções Personalizadas).
1. Copie todo o texto do arquivo `CODE_Skill_v3.md`.
2. Vá nas configurações do seu perfil na IA e procure por "Personalização" ou "Instruções Personalizadas".
3. Cole o texto na caixa "Como você gostaria que a IA respondesse?".
4. Salve. A partir de agora, a sua IA sempre saberá fazer confeitaria de luxo.

### Método 3: Para usuários do Antigravity (ou softwares locais)
Se você utiliza interfaces locais como o **Antigravity** conectado à API do Gemini:
1. Faça o download do arquivo `CODE_Skill_v3.md` para o seu computador.
2. No seu software, vá até a aba de configuração de Persona, Agente ou "System Prompt".
3. Aponte o software para ler este arquivo ou simplesmente copie e cole o texto na caixa de instruções de sistema.
4. Inicie um novo chat e peça: *"Quero fazer um bolo"*. O agente assumirá o controle.
